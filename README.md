# Notepad

A place where I can store useful bookmarks and notes that are searchable.

## Usage

### Start

Will start the parcel dev server with HMR and open the browser.

```sh
yarn start
```

### Build

Compiles the code for production

```sh
yarn build
```

### Lint

Command line linting

```sh
yarn lint
```

### Format

```sh
yarn format
```
